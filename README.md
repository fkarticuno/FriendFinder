# FriendFinder
This app, hosted on Heroku uses:
* "dotenv": "^8.2.0"
* "express": "^4.17.1"
* "path": "^0.12.7" 

To accomplish the storage and [potential] comparrison logic of people with their photos and responses to 10 survey questions.
